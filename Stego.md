### Zsteg

    gem install zsteg
useful commands

    zsteg -a file       # Runs all the methods on the given file
    zsteg -E file       # Extracts data from the given payload

___
### Stegcracker

    pip3.6 install stegcracker

    stegcracker <file> [<wordlist>]
